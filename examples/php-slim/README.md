# php-slim

PHP API server built with [Slim Framework](https://www.slimframework.com/), scaffolded by [Theo](https://usetheo.dev).

## Development

```bash
composer install
composer start
```

The server starts on `http://localhost:8000`.

## Endpoints

- `GET /` — Hello message
- `GET /health` — Health check

## Deploy

```bash
theo deploy
```
